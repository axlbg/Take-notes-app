# Backend - Take Notes App

- Node.js - NestJS

## Endpoints

### Category

- GET /category

- GET /category/:id

- POST /category
    { "name" : "Category name" }

### Notes

- GET /notes/active

- GET /notes/archived

- POST /notes

- PUT /notes/:id

- DELETE /notes/:id

- PATCH /notes/:id/archive

- GET /notes/filterByCategory/:categoryId'
